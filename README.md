Customer Segmentation using K-Means & Flask
This project is a Machine Learning + Web Application that segments customers into meaningful groups based on their Annual Income and Spending Score. The system not only predicts customer type but also provides business marketing recommendations.
Machine Learning Model
Algorithm: K-Means Clustering
Steps performed:
Data preprocessing
Feature scaling using StandardScaler
Elbow method to determine optimal clusters
Model training with 5 clusters
Model saved using Pickle
Technologies Used
Python
Pandas, NumPy
Scikit-learn
Matplotlib, Seaborn
Flask
HTML, CSS
